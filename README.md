#HostelBuddy
Because hostel life needs a buddy.

#About the App
HostelBuddy is an Android application designed to make hostel life easier for students.
It helps students manage common hostel-related activities in a simple and user-friendly way.

#Features
Easy to use interface
Student-friendly design
Helps manage hostel-related information

#Tech Stack
Kotlin

#How to Run 
1.Clone the repository
2.Open the project in Android Studio
4.Run the app on an emulator or real device
